# Parameterized Verilog ALU

A synthesizable parameterized ALU written in Verilog.

## Features

- Parameterized data width
- ADD
- SUB
- AND
- OR
- XOR
- Shift Left
- Shift Right Logical
- Shift Right Arithmetic
- SLT
- SLTU

